# Практическая работа №6
## Задание: Разработка приложения
Концепция приложения: универсальный конвертер длины между разными единицами.

## Проделанная работа:
1. Создана разметка с блоком ввода, `Spinner` для выбора единицы и таблицей результатов в `ScrollView`.
2. Реализован список единиц (мм, см, м, км, дюйм, фут, ярд, морская миля, световой год) через `enum` и `ArrayAdapter`.
3. По нажатию "Перевести" значение переводится в метры и вычисляются все остальные единицы.
4. Добавлены обработка ошибок ввода и отображение сообщения.
5. Настроено форматирование результата (убираются лишние нули, для больших/малых значений используется экспоненциальный формат).

## Итоги
Приложение конвертирует длины в несколько единиц и закрепляет работу со обработкой ввода и выводом результатов.
